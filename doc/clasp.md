## Install CLASP(Command Line Apps Script Projects)

您可以選擇使用[CLASP](https://github.com/google/clasp)來使遠端gs檔案與本機端同步

可以參考此教學

Update npm:
> npm install -g npm@9.1.2

> npm install -g @google/clasp


```
clasp login
clasp logout
```
